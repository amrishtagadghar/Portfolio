# Balance Portfolio

Initial implementation of a design-first portfolio based on the requirements in `initial.txt`.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Run
1. Install Node.js 20+ and npm.
2. Install dependencies:
   - `npm install`
3. Start dev server:
   - `npm run dev`

## Notes
- Content is currently seeded in `lib/content.ts`.
- Routes are ready for CMS integration (Sanity/Contentful) in a next phase.
